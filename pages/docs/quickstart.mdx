# Angular Form Engine quickstart

This site is the reference for [`@openmrs/ngx-formentry`](https://github.com/openmrs/openmrs-ngx-formentry), the optional Angular Form Engine. Use it only when your O3 distribution explicitly includes `@openmrs/esm-form-entry-app`.

The OpenMRS Reference Application uses the React Form Engine by default. Its built-in O3 Form Builder creates and previews forms with React, not Angular. For that workflow, use the [canonical O3 Forms guide](https://o3-docs.openmrs.org/en-US/docs/forms-in-o3/).

There is currently no supported visual builder for the Angular engine. Author Angular-targeted schemas as JSON, validate them against the [standard O3 form schema](https://github.com/openmrs/openmrs-contrib-json-schemas/blob/main/form.schema.json), and test them in an O3 distribution configured with the Angular engine. Schema conformance alone does not guarantee feature parity with the React engine.
