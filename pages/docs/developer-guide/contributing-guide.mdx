## Dependencies

Requires [Node.js v12](https://nodejs.org/es/blog/release/v12.13.0/) or lower. Clone the [repo](https://github.com/ampath/ngx-openmrs-formbuilder) and run `npm install` in the root of your directory.

## Building

Run `npm run build`. Alternatively, run `npm run build:prod` to create an optimized production build.

## Development

Run `npm run start`. This will fire up a development server on `https://localhost:4200`.
